# **Sync_P2P** 

A P2P file synchronizer in ANSI C

## Dependencies
-   [pthread] (http://en.wikipedia.org/wiki/POSIX_Threads) POSIX Threads
-   [jansson] (http://www.digip.org/jansson/) JSON Parser 

## License
-   GPL v3

## Installation
```bash
git clone git@github.com:malderete/sync_p2p.git
cd Debug
make sync_p2p
```
